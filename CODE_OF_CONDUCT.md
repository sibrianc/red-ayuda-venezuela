# Código de conducta

## Nuestro compromiso

Este es un proyecto humanitario. Nos comprometemos a ofrecer una experiencia libre de
acoso para todas las personas, sin importar edad, identidad, discapacidad, origen,
nacionalidad, religión, orientación sexual o situación migratoria.

## Comportamiento esperado

- Trata a las demás personas con respeto y empatía; recuerda que detrás de cada reporte
  hay personas en una emergencia.
- Cuida los datos sensibles: nunca compartas ni publiques información privada de víctimas
  o reportantes.
- Acepta la crítica constructiva y prioriza el bienestar de las comunidades afectadas.

## Comportamiento inaceptable

- Acoso, lenguaje o imágenes de odio, ataques personales.
- Exponer información privada de terceros sin permiso (doxxing).
- Usar el proyecto o sus datos para vigilar, perseguir o dañar a personas.

## Aplicación

Reporta conductas inaceptables al responsable del proyecto en
**carlosssibrian@gmail.com**. Todas las quejas se revisarán de forma confidencial. Quien
incumpla este código puede ser advertido, vetado temporal o permanentemente.
